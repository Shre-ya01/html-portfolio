About Me

Hello! I am a beginner web development student currently learning HTML. This portfolio is one of my first projects, created to practice the fundamentals of web design and page structure.

In this project, I developed a portfolio website that includes a Movie Ranking Project, a Birthday Invitation Project, and separate About Me and Contact Me pages. Through this project, I learned how to use headings, paragraphs, images, links, and multiple HTML pages to build a simple website.

This project has helped me strengthen my understanding of HTML and has motivated me to continue learning CSS and JavaScript to create responsive and interactive websites in the future.
